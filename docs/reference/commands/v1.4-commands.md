# v1.4 Command Reference

## Knowledge

```bash
soturail knowledge estimate <paths...>
soturail knowledge compile <paths...> --name <name>
soturail knowledge update <name> <paths...>
soturail knowledge verify <name>
soturail knowledge list
```

## Evidence

```bash
soturail evidence collect
soturail evidence verify
soturail evidence report
```

## Agent QA

```bash
soturail eval dataset init
soturail eval dataset run
soturail eval golden
soturail eval regression
soturail eval report
```

## Skill Rail 2.0

```bash
soturail skills template <domain>
soturail skills build <paths...> --name <name>
soturail skills fold-in <skill> <paths...>
soturail skills lint
soturail skills eval
soturail skills report
```

## Tasklets

```bash
soturail tasklet create <name>
soturail tasklet list
soturail tasklet run <name> --dry-run
soturail tasklet export <name>
```

All commands are local-first. Knowledge and evaluation are deterministic; evidence does not invent verification; tasklet execution is simulation-only.
