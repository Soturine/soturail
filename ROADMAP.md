# Roadmap

## v0.1.0 - MVP Core Foundation

- Local `.soturail` workspace
- Heuristic Repo Map
- Progressive file reader
- Safe tee-stream command runner
- Raw log expansion
- Spec-Driven Development artifacts
- Git-linked local memory
- Stable block cache normalizer
- Honest local stats

## v0.2.0 - Context Payload and Reducer Depth

- Add `soturail context` for prompt-ready payload generation.
- Improve `.gitignore` compatibility while keeping no native dependencies.
- Add richer reducers for TypeScript, package managers and CI logs.
- Add manual provider metadata import for real cache-hit reporting.
- Add memory approval workflow for stable cache inclusion.
- Add Windows PowerShell command-shape tests and more shell quoting coverage.

## Later

- Optional Rust or Go runner behind `NativeRunnerAdapter`.
- Pluggable reducers and policy packs.
- Repository graph enrichment with real parsers when dependency constraints allow.
- UI or TUI review surface for raw logs, specs and memory.
