SotuRail v0.2.1 is the first public npm release.

## Install

- npx soturail --help
- npm install -g soturail
- soturail --help

## Highlights

- Published on npm.
- Local-first Context OS for AI coding agents.
- Reversible terminal compression and raw log recovery.
- Progressive repo reading.
- Spec-Driven Development workflow.
- Agent response compression.
- Knowledge-to-Rules ingestion.
- Prompt-cache-friendly payload organization.
- Local benchmarks and metrics.
